Styling Links

This page shows a few of the common ways in which you might want to style links. To see how each is accomplished, right click on the page and select "View Page Source" on the context menu.

Site Navigation

Your main site navigation links are typically marked up as links inside an unordered list, inside a nav element. Using CSS, we can transform those into something that looks like a navigation bar.
