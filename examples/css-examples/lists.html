Styling Lists

Besides styling lists of navigation links to look like a navbar or tabs, there are a few other ways in which developers often alter the look of lists. This page shows you a few examples. Like the others, right-click on the page and select "View Page Source" to see how this is done.

Traditional Outline

You can alter the list style at different levels to get a traditional topic outline look: upper-case Roman numerals at the first level; decimals at the second level; lower-case alpha at the third. Use child selectors to target different levels.

  1. Introduction
    1. Topic 1
    2. Topic 2
  2. Body
    1. Point 1
      1. sub point 1.1
      2. sub point 1.2
    2. Point 2
      1. sub point 2.1
      2. sub-point 2.2
  3. Conclusion
    1. Wrap up 1
    2. Wrap up 2

Images for Bullets

You can choose between a few different built-in bullet shapes, but you can also use your own icons as bullets as well. The best way to do this is using a background-image instead of list-style-image, as you can control the alignment more precisely. Just set the list-style to none, add the background-image (no repeat, left center), and add some padding-left to push the list item text over.
