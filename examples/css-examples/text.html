Styling Text Example Page

This page shows a number of common ways in which you might want to style your text. For example, this first paragraph has a drop cap, and the heading above uses a subtle shadow. To see how this is accomplished, right click on the page and choose "View Page Source" from the context menu in either Firefox or Chrome (or most other browsers as well). All styles are defined in a <style> element in the <head> section of the document, which makes it easier for you to see the styles and HTML source in one place. Normally, you'd want to split your styles into a separate file and then link that file to the page using a <link> element.

Instead of showing you every single property that controls the appearance of text and paragraphs (see the CSS text formatting reference for that), I'll use this page to show you how to do a few common things you might want to do that involve multiple settings.

Blockquotes

Browsers simply indent blockquotes by default, but it's common to add some additional formatting to call them out. Let's add a background color, left border, and some padding.

Lorem ipsum dolor sit amet, consectetur adipisicing elit. Fugiat, consequatur, quae maxime quibusdam ducimus quas rem dicta optio dolor perspiciatis quo earum asperiores non facilis aspernatur libero voluptas aliquam magnam.

Inline Quotes

Browsers will wrap inline quotes (<q>) with language-appropriate quotation marks, but you can override this to offer nicer-looking curly quotes using the quotes property.

As Abraham Lincoln was often fond of saying, I don't think people quote me correctly on the Internet!

Alerts

Sometimes you want to make a paragraph stand out, especially if it provides important information to the reader. Use background colors, borders, and even a background icon to do that.

This is a warning. Pay attention to me!

Revision Marks

When important documents are revised, it's common for authors to display what text was removed, what text was added, and what text had open questions or comments. These can be achieved by wrapping the relevant bits of text in del, ins, and mark elements. Browsers have default formatting for these, but you can alter then in CSS. This text was deleted. This text was added. This text has a comment assoicated with it.
