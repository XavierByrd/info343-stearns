Styling Boxes

This page shows you a few of the common ways that developers style boxes on the page, that is, the areas that surround block elements. To see how these effects are accomplished, right-click on the page and choose "View Page Source" from the context menu.

The Box Model

Any element that displays as a block is contained within a box that you can style in several interesting ways. In order to understand this, however, you first need to visualize the CSS box model:

margin
border
padding
content

The margin value separates the box from other boxes in the same container. The border value defaults to 0, but can be set to any thickness and color you want. And the padding value controls how much space is betwen the border and the content of the element. Each of these values can be set for all sides at once, or controlled independently (e.g., set a bottom border, but not a left, top, nor right).

Shaded Backgrounds

In addition to a border, boxes can also have a background-color, which fills up to the inside edge of the border (including the padding). For example, important feedback for the user is often displayed in a paragraph with a pale yellow background-color and a slightly darker yellow border, and some padding around the content.
