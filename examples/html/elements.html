Most Commonly Used

h1–h6

Content is a section heading, from level 1 through 6

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>
<h4>Heading 4</h4>
<h5>Heading 5</h5>
<h6>Heading 6</h6>

p

Content is normal paragraph text

<p>This is a paragraph</p>

This is a paragraph

a

Content is a hyperlink

href attribute is the target URL. Add target="_blank" attribute to automatically open the target page in a new browser tab.

<a href="http://google.com">Google</a>
Google 
<a href="http://google.com" 
target="_blank">Google</a>
Google

img

Indicates that an image should appear here

src attribte is URL to the image file; alt is a textural description for screen readers and shown if the file can't be loaded. Note that this element doesn't have an end tag.

<img src="kitten.jpg" alt="cute kitten">

br

Indicates that a manual line break should appear here

<p>first line <br> second line</p>
<p>next paragraph</p>

second line
