Most Commonly Used Body Elements

When writing body content, these are the elements you'll use the most

a
Content is a hyperlink to the URL specified in the href attribute.
The content may contain other elements (e.g., img), and all elements will be part of the link. Set the target attribute to "_blank" to automatically open the URL in a new browser tab.

img
The specified image should be displayed here
Use the alt attribute to specify an alternate textural description. This is used by screen readers, and displayed if the image file cannot be found.

ol, ul, li
Ordered and Unordered lists

em
Content should be emphasized
Although most agents will simply display the content in italics, use this for the meaning it conveys, not the default formatting it achieves.

HTML should convey meaning not desired formatting.

strong
Content is important
Although most agents will simply display the content in bold text, use this for the meaning it conveys, not the default formatting it achieves.

When getting dressed, remember this rule: socks, then shoes!

br
Inserts a manual line break
A br will cause a manual line break within the current block, but not the start of a new block. This is useful when you don't want to start a new paragraph, but you need a line break to occur.

blockquote, q
Content is a lengthy, or short quote
Use blockquote for long quotes that should be set off from the main content, and q for short, inline quotes.
