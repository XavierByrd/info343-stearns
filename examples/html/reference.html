Sectioning Elements

These elements are used to indicate major sections of the document. They affect the outline generated for a page, which is used by screen readers, and may be used by other agents to provide a table of contents. They don't typically receive any default formatting, but you can use CSS to format these sections in different ways.

article
Content is a cohesive unit that could be syndicated or displayed outside this page
Use this for the main part of your page, especially if that content is like an article that could be syndicated and appear on its own.

My Content

This is my article.

nav
Content is a set of navigation links
Use this for your navigation links so that they don't become part of the outline, and screen readers can skip over them if desired.
  • Home
  • About
aside
Content is a side comment, or not part of the main article
Use this for pull-quotes, or other content that is tangential or unrelated to the main article.

"I don't think I'm quoted very accurately on the Internet" - Abraham Lincoln

section
Content is a separate section of the main article
The meaning of this is rather vague in the HTML specification, so most people use this only for significant sections that you might want to treat separately. Sections should have a heading element.
