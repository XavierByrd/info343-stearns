Multi-Column Layout Examples

This page demonstrates three approaches to multi-column, grid-based layouts: using floats; using the display setting inline-block; and using the display setting table-cell. The columns have explicit percentage-based widths, but their height is left to auto, so the column box will expand vertically to display all of its content.

If you want to constrain the height as well, you'd then have to decide what to do when the content overflows the fixed box size. The overflow property in CSS can be used to clip content that overflows, or display internal scroll bars.
