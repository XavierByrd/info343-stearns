Helpful Resources

Links to helpful tools and references

Color

The Color Scheme Designer
Handy tool for generating different types of color schemes given a starting color
Adobe® Kuler
Another tool for generating color schemes, with lots of example themes to choose from
Colour Lovers Palette Gallery
A large gallery of color palettes contributed and voted upon by artists
Somacon Color Chart
A nicely-organized general color reference

Textures

Subtle Patterns
A collection of free subtle textures, useful for lightly-colored body backgrounds
Lost and Taken Textures
A collection of high-resolution textures, most of which are more "grunge" than the subtle ones above

Icons

Icon Finder
A fabulous collection of icons, most of which are free and come in a variety of sizes

Stock Photos

Morgue File Free Photo Archive
A handy archive of free stock photos

Fonts

Google Fonts
A collection of free custom fonts that you can reference directly in your stylesheets or HTML pages. Click on the 'Quick-use' button next to any font to get the necessary HTML/CSS.
Lorem Ipsum Generator
A tool for generating any amount of "lorem ipsum" text. Web developers and designers use this to add dummy text to their pages in order to see how their font and page layout choices will look. If you use Sublime Text as your text editor, you can also generate this in the editor itself: type 'lorem' (all lower case) and then hit tab.

Browser Support

Can I Use...
A site that tracks support for different CSS and HTML features across all major browsers (including mobile versions)

CSS References

CSS Selectors Reference
A handy table showing all the various kinds of CSS selectors with examples and descriptions of what they select
CSS Properties Reference
A complete reference for all the various CSS properties, organized by function
CSS Units Reference
Short reference of the various unit measurements supported in CSS and their suffixes

HTML References

General HTML 5 Reference
A general reference for HTML 5 elements, organized by function. See also the alphabetical reference
Global Attributes Reference
A general reference for the global HTML attributes, which may be used on any element in the body section of your document
HTML Character Entities
A reference for all the various supported character entities. Use these to encode reserved characters and other symbols

JavaScript References

JavaScript and the DOM Reference
A general reference for the core JavaScript language and the Document Object Model (DOM)

Validation Tools

W3C HTML Validation Service
Use this to validate your HTML pages. Browsers will often let you get away with invalid HTML, but this service will tell you if something is wrong in your page.
W3C CSS Validation Service
Use this to validate your CSS pages. Browsers will just ignore invalid content in your CSS pages without any warnings, so you if you are having troubles getting your CSS to work, try validating your stylesheet to make sure you don't have an error.
JS Lint
Use this to detect potential problems in your JavaScript. This is a very aggressive lint tool, however, so take its advice with a grain of salt.
