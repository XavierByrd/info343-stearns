Homework Assignments

Requirements and supporting materials

Congratulations! You've just been hired by the hottest new pizzeria, Dawg Pizza, to build their web site. You will be working with the owner and founder, Michael Chen, to build a site that will both he and his target customers will love.

You will build this site in four phases, adding complexity and features each time. We will assign these phases the version numbers 1.0, 2.0, 3.0, and 4.0. The requirements and due-dates for each version are listed below, and you'll submit your completed versions via the respective Canvas assignment.

For each of these versions, Michael will supply you with his list of must-have and optional requirements, plus some images and text copy you should include in the pages. If you want to ask clarifying questions, feel free to post them to our "Questions for Pizzeria Owner" discussion board in Canvas. The design of the pages will be up to you, however, so be creative and design something that both the owner and his target customers will find engaging, intuitive, and accessible. There is no one right way to build these sites, but we will examine your source to make sure you've used the appropriate HTML, CSS and JavaScript constructs for your particular solutions.

In order to get a full 10 points for the "completeness" portion of your grade, you must meet all the "must-have" requirements and at least one of the optional requirements. If you only do the must-have requirements, you'll get only 9 points on that portion of your grade. See the rubric displayed below each homework assignment in Canvas.

When you complete a version of your site, publish it to a new folder on your iSchool student web server and submit the URL via the Canvas assignment. If you don't link to the new folder from anywhere, it will remain an "island" on the web and no one else besides your instructor and TA will be able to find it. If you want to publish a link to the folder on the Internet, feel free to do so, but there is no requirement to do so.

Although you are not explicitly required to use a version control system while developing this site, I would strongly recommend that you do so. A version control system like Git/GitHub will not only help you manage changes and recover from serious mistakes, but also keep your code backed-up in an offsite location in case your computer is lost, stolen, or broken. This is also a great opportunity to practice using these tools so that you are ready to jump into a professional web development team.

These are individual homework assignments, and this should be entirely your own work. Do not work on these sites together, and do not share code with each other. You may use open-source CSS frameworks and JavaScript libraries developed by others, but your own code must be your own code. If you get stuck on a technical question, you may ask for help on our "Technical Help" Canvas discussion board, but don't expect anyone to reply at 10pm on the night before the assignment is due.

Version 1.0 Due by Oct 8

Michael is working hard to get his pizzeria ready for their grand opening on October 8th, and he wants some kind of web presence by then. Below are his specific instructions. Remember that he isn't a web developer, so you'll need to interpret what he says and use the appropriate markup and styles. Color choices and page layout are entirely up to you unless otherwise stated. If you draw a blank on how it should look, check out some other pizzeria web sites and see what they did.

Hey there! I'm excited that you're going to build my site. We're set to open on October 8th, and I'd like to have something up on the web by then. I know that doesn't give you much time, so for this first version, let's keep it kind of simple.

Since you're the expert on this stuff, I'll leave the page design to you. Make it look as cool as you can. My target customers are university students like you, so put together something that would make you want to order pizza from us!

Here are my must-haves:

  • There should be three pages: home, menu, and about us.
  • All pages should have the same set of navigation links to get from one page to the other.
  • All pages need to include a small version of our logo, and clicking on it should take you back to the home page.
  • The home page should have this on it:
    • The name of our restaurant (Dawg Pizza) prominently displayed
    • This short description: "Awesome wood-fired pizza with fresh ingredients baked to order. Come in and eat, or call us for delivery!"
    • Our phone number (206-555-1212) and address (4306 University Way NE, Seattle, WA 98105)
    • Our email address (dawgpizzaseattle@gmail.com) and make it so that when you click it, it opens a new email message to us.
    • A link to our twitter feed (@DawgPizzaSea)
    • This picture of our pizza (it's big, so you might need to make it smaller)
  • The menu page should list out our various menu options, and have this picture of our pizza on it.
  • The "about us" page should have this on it:
    • This text about our restaurant
    • A picture of our wood-fired pizza oven

And if you get time, I'd also love to have one or more of these added:

  • Add a "Jobs" page that says we are now hiring kitchen staff as well as delivery drivers.
  • Make it so that if someone clicks on our address, it takes them to Google Maps showing that address.
  • Add a footer to all the pages with this content: "Call us for delivery: 206-555-1212"
