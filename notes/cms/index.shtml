  3. Content Management Systems (CMSs)
    1. CMSs manage relatively unstructured content through a publication and archival process
    2. CMS may support several kinds of content, but each kind typically consists of a blob of text with accompanying meta-data
    3. Publication process tracks the content through several stages: create, edit, review, publish, collaborate, archive
    4. Security can be applied throughout the process so that only certain people are allowed to do a given task at a given time in a given context (e.g., an author might be able to create a new document, but only publishers are allowed to publish, and only managers can archive)
    5. Content can be categorized for easy browsing
    6. Content can be searched for easy finding
